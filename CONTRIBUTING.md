# 🤝 Contributing to HexVoice-AI

Thank you for your interest in contributing to **HexVoice-AI**! 🎉
We welcome contributions from everyone to make this project better.

---

## 📌 How to Contribute

### 1️⃣ Fork the Repository

Click on the **Fork** button at the top right of this repository.

---

### 2️⃣ Clone Your Fork

```bash
git clone https://github.com/Ronit049/HexVoice-AI.git
```

---

### 3️⃣ Create a New Branch

```bash
git checkout -b feature/Ronit049-patch-1
```

---

### 4️⃣ Make Your Changes

* Add new features
* Fix bugs
* Improve documentation
* Optimize performance

---

### 5️⃣ Commit Your Changes

```bash
git commit -m "Add: your meaningful commit message"
```

---

### 6️⃣ Push to GitHub

```bash
git push origin feature/Ronit049-patch-1
```

---

### 7️⃣ Create a Pull Request

Go to your forked repository and click on **New Pull Request**.

---

## 🧑‍💻 Contribution Guidelines

* Follow clean and readable code practices
* Write meaningful commit messages
* Keep your changes focused and minimal
* Test your code before submitting
* Update documentation if needed

---

## 🐛 Reporting Issues

If you find a bug or want to request a feature:

1. Go to the **Issues** tab
2. Click on **New Issue**
3. Describe the problem clearly

---

## 💡 Suggestions

We welcome ideas for:

* New voice commands
* AI integration
* UI/UX improvements
* Performance enhancements

---

## 📜 Code of Conduct

* Be respectful and inclusive
* Avoid offensive language
* Collaborate positively with others

---

## 🙌 Thank You

Your contributions help improve this project and make it more useful for everyone!

Happy Coding! 🚀
